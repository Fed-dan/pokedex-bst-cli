# Pokedex BST CLI

An educational Python command-line project for managing Pokemon Pokedex owners using a Binary Search Tree.

## Features

- Load Pokemon data from a CSV file
- Create and delete Pokedex owners
- Store owners in a Binary Search Tree ordered by owner name
- Add Pokemon to an owner's Pokedex
- Release Pokemon by name
- Evolve Pokemon
- Display Pokemon with filters
- Traverse owners using:
  - BFS
  - Preorder
  - Inorder
  - Postorder
- Sort owners by number of Pokemon

## Technologies

- Python
- CSV file handling
- Binary Search Tree
- Recursion
- Command-line interface
